# Satellite image modality

This directory contains image-native Sentinel-2 L2A samples generated by `src/download_sentinel2_chips.py`.

Expected structure after a successful workflow run:

```text
data/satellite/
├── README.md
├── metadata.csv
├── manifest.json
└── chips/
    ├── sentinel2_quarterly_2022.npz
    ├── sentinel2_quarterly_2023.npz
    ├── sentinel2_quarterly_2024.npz
    └── sentinel2_quarterly_2025.npz
```

Each NPZ shard contains four-band tensors (`B02`, `B03`, `B04`, `B08`), local valid-pixel masks derived from the Scene Classification Layer, and sample IDs. The samples are quarterly 64 x 64 images covering approximately 6.4 km around each province market centroid.

Read the full technical specification in [`docs/datasets/SENTINEL2_IMAGERY.md`](../../docs/datasets/SENTINEL2_IMAGERY.md).

Important: these patches represent regional context around market centres. They are not verified chili or shallot field observations.
